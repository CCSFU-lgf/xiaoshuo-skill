# xiaoshuo-skill 小说创作辅助技能

这是一个为小说平台作者设计的创作辅助技能，帮助作者进行小说创作前的准备工作。

## 功能特性

### 1. 书名检查
- 检查书名是否符合平台规则和内容安全要求
- 识别敏感词、特殊字符等违规内容
- 提供修改建议

### 2. 标签定位
- 根据书名智能匹配作品标签
- 支持主分类、主题、角色、情节四个维度
- 提供匹配置信度

### 3. 随机生成
- 为没有明确目标的用户提供随机标签组合
- 支持按分类、主题生成
- 确保组合不重复

### 4. 创作指导
- 提供基于标签的开篇建议
- 推荐同类热门作品参考
- 提供写作技巧和去AI味建议

## 安装与使用

### 1. 安装

```bash
python install.py
```

### 2. 启动交互式界面

```bash
python start.py
```

### 3. 直接使用

```python
from xiaoshuo_skill import XiaoShuoSkill

# 创建技能实例
skill = XiaoShuoSkill()

# 检查书名
result = skill.check_book_name("《重生之都市修仙》")
print(result)

# 匹配标签
result = skill.match_tags("《豪门总裁的替身新娘》")
print(result)

# 随机生成
result = skill.generate_random_tags(count=3)
print(result)

# 创作指导
result = skill.get_writing_guidance(
    main_category="豪门总裁",
    themes=["现代言情", "豪门世家"],
    characters=["总裁", "大小姐"],
    plots=["契约婚姻", "先婚后爱"]
)
print(result)
```

### 4. 运行示例

```bash
python example.py
python demo.py
```

## 知识库内容

### 平台规则
- 基本要求：原创、标题相符、黄金三章、语言流畅、章节结构
- 内容红线：政治敏感、色情暴力、违法内容、三观问题、未成年人保护
- 写作技巧：黄金三章、爽点密集、节奏紧凑、钩子结尾、人设鲜明、打脸套路

### 内容安全须知
- 禁止内容：8大类禁止发布的内容
- 审核不通过原因：10种常见审核不通过原因
- 处罚说明：违规作品的处罚措施

### 作品标签
- 主分类：21个分类（女频悬疑、古风世情、科幻末世等）
- 主题：25个主题（古言权谋、悬疑恋爱、纯爱等）
- 角色：40个角色（总裁、忠犬、全能等）
- 情节：92个情节（代嫁代娶、攻略反派、风水秘术等）

## 使用场景

### 场景1：新手作者
```
用户：我想写小说，但不知道写什么
技能：为您生成3个随机标签组合...
```

### 场景2：有明确想法
```
用户：我想写一本叫《重生之都市修仙》的小说
技能：正在检查书名... 书名符合规范
正在为您匹配标签...
推荐标签：现言脑洞 + 幻想言情 + 大佬 + 重生
```

### 场景3：需要创作指导
```
用户：我选择了豪门总裁类，怎么写开篇？
技能：豪门总裁类开篇建议：
1. 第一章：制造强烈冲突，如契约婚姻、替身新娘
2. 第二章：展现男主强大气场和女主独特魅力
3. 第三章：设置悬念，如女主隐藏身份、男主秘密
```

## 注意事项

1. 所有创作必须符合《内容安全须知》和《平台规则》
2. 避免AI味，多参考热门作品
3. 标签组合要合理，避免矛盾
4. 开篇要吸引人，遵循黄金三章法则
5. 每章保持1500-3000字，节奏紧凑
